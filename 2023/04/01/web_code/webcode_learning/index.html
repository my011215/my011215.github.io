DOM 生命周期

  1. DOMContentLoaded-- 完全加载构造 DOM 树
  2. load -- 加载额外图片样式等资源
  3. beforeunload 事件 —— 用户正在离开：我们可以检查用户是否保存了更改，并询问他是否真的要离开。
  4. unload 事件 —— 用户几乎已经离开了，但是我们仍然可以启动一些操作，例如发送统计数据。
